## ğŸ¯ What It Does

DataOps Commander is a lightweight data processing tool built for sales and marketing teams. Upload messy Excel/CSV files, get clean insights - no coding required.

Built this while working at Property Pistol to automate repetitive data cleaning tasks that were eating up hours every week.
